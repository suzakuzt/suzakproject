# P2 QA 验收 Agent 任务卡

# 项目背景

项目：高考考运摇签 + 牛肉福利领取 H5 活动  
页面：P2 今日考运签结果页 / 解签入口页

## 你的角色

你是本项目的 QA Agent，只负责验证 P2 页面逻辑是否符合当前主设计图整理口径。

## 必须测试

1. 从 P1 进入 P2 后，能展示完整结果页结构；
2. 顶部展示「六月考运签」；
3. 签型标签正常展示；
4. 主签文两列竖排正常展示且支持动态内容；
5. 左右宜 / 忌内容正常展示且支持动态内容；
6. 点击「问小璞」后可进入解签流程；
7. 点击「点击分享海报」后可进入海报流程；
8. 结果页接口失败时有异常提示；
9. 解签接口失败时有异常提示；
10. 海报生成失败时有异常提示；
11. 页面不出现再抽一次、商品卡、优惠券、企微二维码等未确认模块。

## 建议测试场景

| 测试场景 | 预期结果 |
|---|---|
| 正常结果页数据返回 | 页面正常渲染 |
| 主签文字段替换 | 页面对应内容变化正确 |
| 宜 / 忌字段替换 | 页面对应内容变化正确 |
| 点击问小璞 | 进入解签入口 |
| 点击分享海报 | 进入海报入口 |
| 结果页接口失败 | 提示加载失败并可重试 |
| 解签接口失败 | 提示解签失败 |
| 海报生成失败 | 提示海报失败 |

## 验收标准

1. 页面核心内容完整可见；
2. 交互路径清晰；
3. 错误态处理完整；
4. 页面未出现未确认模块；
5. 与当前主设计图结构保持一致。
