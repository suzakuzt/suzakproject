# P2 H5 前端开发提示词

你是本项目的 **H5 前端开发 Agent**。

## 当前任务

只实现 **P2 今日考运签结果页 / 解签入口页**。

## 重要限制

1. 只处理 P2，不开发活动规则页或其他页面的完整业务。
2. 问小璞解签流程、分享海报流程可先做路由占位、弹窗占位或页面占位。
3. 不要新增未确认的业务规则。
4. 不要改动与 P2 无关的页面。
5. 不要私自改文案。

## 输入资料

- `P2_H5前端_Agent任务卡.md`
- P2 设计图
- 当前项目目录结构
- 当前前端技术栈说明

## P2 已确认规则

1. 本页为用户完成摇签后的结果展示页。
2. 顶部固定展示「六月考运签」。
3. 签型标签区展示如「金榜题名签 · 上上签」，需支持动态配置。
4. 中间两列竖排主签文为页面核心结果区域，需支持动态渲染。
5. 左侧展示「宜」，右侧展示「忌」，后面的内容需支持动态渲染。
6. 中部固定展示「今日解签」与「点击下方按钮，查看今日解签内容」。
7. 主按钮固定为「问小璞」。
8. 底部次操作固定为「点击分享海报」，并保留下划线视觉。
9. 当前页不默认出现再抽一次、奖励领取、商品卡、企微二维码等模块。

## 需要实现

1. 按设计稿还原 P2 结果页。
2. 实现签型标签、主签文、宜 / 忌内容的动态渲染。
3. 实现点击「问小璞」后进入解签入口。
4. 实现点击「点击分享海报」后进入海报入口。
5. 支持结果页加载态、失败态、解签加载态、海报生成态。
6. 保留后续接接口的位置，不要把业务写死。
7. 完成后请运行项目现有的 `lint / build / test` 命令，并说明结果。
