你是本项目的 **H5 前端开发 Agent**。

## 当前任务

只实现 **P4 AI 解签结果页 / 福利承接页**。

本页基于 P4 主设计稿开发，用于承接 P2「问小璞」动作，展示 AI 解签结果，并引导用户点击「领取专属福利」。

## 接口占位建议

### GET /api/explain/detail

用于进入 P4 时获取 AI 解签结果。

当前实现由后端代理 DeepSeek `/chat/completions` 生成 AI 解签文案，前端不接触 API Key。后端配置项包括 `DEEPSEEK_API_KEY`、`DEEPSEEK_BASE_URL`、`DEEPSEEK_MODEL`、`DEEPSEEK_REASONING_EFFORT`、`DEEPSEEK_TIMEOUT_SECONDS`、`DEEPSEEK_MAX_TOKENS`。未配置或失败时返回本地兜底文案。

页面 loading 展示「AI 解签中」和 `thinkingProcess` 对应的短句/点状动效；成功后只展示最终签文 `explainLines`、`themeText` 和商品福利，不展示模型内部推理。

建议返回：

```json
{
  "activity_id": "gaokao_lucky_sign_2026",
  "session_id": "string",
  "draw_id": "string",
  "explain_id": "string",
  "sign_type": "金榜题名签",
  "sign_level": "上上签",
  "title": "AI解签结果",
  "explain_lines": [
    "锦绣前程，步步生花",
    "实力如锦，终成佳绩",
    "心之所向，金榜题名"
  ],
  "theme_text": "高考好运 × 牛气补给",
  "product": {
    "product_id": "string",
    "product_name": "和牛 · 锦绣前程板腱",
    "product_image": "https://example.com/product.png"
  },
  "benefit": {
    "benefit_id": "string",
    "claim_status": "unclaimed",
    "claim_button_text": "领取专属福利"
  }
}
```

### P5 领券接口

`POST /api/benefit/claim` 不应由 P4 点击时直接调用，而是由 P5 用户输入手机号并点击「去领取」后调用。

请求体必须包含：

```json
{
  "session_token": "string",
  "draw_id": "string",
  "reward_code": "coupon_10",
  "mobile": "13812345678",
  "claim_channel": "h5"
}
```
