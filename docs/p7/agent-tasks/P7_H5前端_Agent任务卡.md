# P7 H5 前端 Agent 任务卡 v1.0

## 当前任务

完成 **P7 活动规则页 / 规则说明页** 的 H5 前端开发。

## 页面定位

P7 是活动规则独立页面，用于说明抽签机会、分享助力、打卡点亮、985 和牛礼盒资格、奖品限制和企微提醒。

## 输入资料

- `P7_活动规则页确认版_Agent交付卡.md`
- P7 主设计稿
- 当前 H5 项目目录结构
- 当前路由配置
- 当前埋点工具 / 页面跳转工具

## 已确认规则

1. 页面标题固定为「活动规则」。
2. 页面副标题固定为「一举高中 · 六月牛气加油签」。
3. 左上角返回按钮返回上一页，无历史栈时回 P1 首页。
4. 每人每日默认 1 次抽签机会。
5. 分享好友参与并完成抽签后，原用户增加 1 次助力进度。
6. 每日通过分享最多获得 3 次额外机会。
7. 按最新口径：每日完成抽签后，自动点亮 1 天进度，每天最多点亮 1 天。
8. 累计点亮 7 天或邀请 5 位好友，可解锁 985 和牛礼盒抽奖资格。
9. 奖品数量有限，具体以活动页面公示为准。
10. 本活动为娱乐互动与祝福活动，不代表任何成绩预测。
11. 底部展示企微二维码和说明文案。
12. P7 不出现抽签按钮、奖励列表、商品推荐、再抽一次、签文结果、AI 解签正文。
